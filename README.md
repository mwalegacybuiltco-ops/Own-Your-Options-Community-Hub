# Own Your Options Collective

Complete community hub rebuild for a fresh GitHub repository.

## Upload To GitHub

Upload every file and folder from this package into the main/root area of the new repository.

Important files:

- `index.html`
- `app.js`
- `cloud.js`
- `styles.css`
- `assets/`
- `.nojekyll`

## Turn On GitHub Pages

1. Open the new repository on GitHub.
2. Open **Settings**.
3. Select **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose the **main** branch and the **/(root)** folder.
6. Select **Save**.

GitHub normally publishes the website within a few minutes.

## Owner App Links

After signing in as April:

1. Open **April Admin**.
2. Select **App Links**.
3. Paste the website links for the Own Your Options App and Coaching Hub.
4. Select **Save app links**.

## Included Features

- Public landing page
- Member sign-in and account creation
- Community posts, comments, and pictures
- Learning library and resource vault
- Live events, challenges, and member directory
- My Apps launch center
- Separate Coaching Hub launch area
- April-only admin back office
- Moderator roles
- Member progress, blocking, archiving, and deletion controls
